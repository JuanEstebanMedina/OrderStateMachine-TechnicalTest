# Order State Machine

Technical challenge project for implementing an order state machine using a Python REST API and a React frontend.

## Project structure

- `backend`: Python REST API
- `frontend`: React frontend

## Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
